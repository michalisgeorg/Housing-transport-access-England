# Housing-transport-access-England
Repository containing the code used in the paper **“The housing market value of public transport accessibility to employment in post-pandemic England”**, published in *Environment and Planning B: Urban Analytics and City Science*.

## Sales listings data (Zoopla plc)
These data are accessed under licence from the Urban Big Data Centre (https://www.ubdc.ac.uk).  
Other UK-based academics may access the data via the UBDC while the licence remains current.

All other datasets used in this repository are open access and are referenced in the code with the relevant links.
